# 學員報到系統 (Google Sheets + Apps Script)

這是一個基於 Google 試算表開發的自動化報到系統，支援 QR Code 掃描與手動輸入，並具備多分頁管理、響應式介面與自動定位功能。

## 🚀 系統特色

*   **多分頁支援**：可在設定中自訂多個報到分頁（如：5/22報到、5/23報到），輕鬆管理不同場次的活動。
*   **響應式切換介面**：
    *   **寬螢幕 (Web App)**：顯示為直覺的水平標籤列（Tabs），支援滑鼠拖曳滾動。
    *   **窄螢幕 (Sidebar)**：自動轉換為下拉式選單（Select），完美契合試算表側邊欄空間。
*   **自動定位跳轉**：在側邊欄報到成功後，試算表會自動跳轉並選取「學員名單」中該學員的對應列（可開關）。
*   **安全管理**：報到訊息模板與分頁清單僅限從 Google Sheets 側邊欄編輯，確保外部網頁無法隨意修改核心設定。
*   **模組化架構**：程式碼採 CSS/JS 抽離設計，易於維護與擴充。

## 📋 工作表規範

### 1) 學員名單
*   **標題列（第 1 列）必須包含**：`學員代號`、`手機`。
*   其餘欄位可自由擴充（例如：姓名、班級、序號...）。
*   系統會以此表的標題名稱作為訊息模板的變數來源。

### 2) 報到分頁 (如：學員報到)
*   由系統自動建立，或手動建立。
*   **A 欄**：系統自動寫入學員代號。
*   **B 欄**：系統自動寫入報到時間。
*   可自行在後方欄位使用公式（如 `VLOOKUP` 或 `XLOOKUP`）來串接「學員名單」的詳細資料。

## 🛠️ 部署指引

### 1. 準備試算表
1.  **建立副本**：檔案 -> 建立副本。
2.  **授權存取**：在「學員名單」中若有 A1 欄位的存取請求，請點擊允許。

### 2. 部署 Apps Script
1.  點擊「擴充功能」 -> 「Apps Script」。
2.  確保專案內包含以下檔案：
    *   `Code.js`: 後端核心邏輯。
    *   `Sidebar.html`, `SidebarStyle.html`, `SidebarScript.html`: 側邊欄模組。
    *   `Bridge.html`: 跨網域通訊橋接。
3.  點擊右上角「部署」 -> 「新增部署作業」。
4.  類型選擇「網頁應用程式」，執行身份為「我」，誰可以存取為「所有人」。
5.  完成後複製 **網頁應用程式網址**。

### 3. 初始化設定
1.  開啟剛剛部署的網頁應用程式網址。
2.  系統若提示「缺少部署 ID」，請確保網址參數正確。
3.  在 Google 試算表上方選單會出現「**額外選單**」，點擊「**顯示報到側邊欄**」。
4.  在側邊欄點擊右下角「**齒輪 (設定)**」，設定您的訊息模板與報到分頁名稱。

## 📖 功能操作說明

### 側邊欄功能
*   **分頁切換**：使用頂部的下拉選單切換目前的目標報到分頁。
*   **自動跳轉**：開啟此開關後，報到成功時試算表會自動滾動到該學員在「學員名單」的位置。
*   **手動報到**：在輸入框輸入代號或手機後點擊「送出」。

### 設定介面 (僅限側邊欄)
*   **訊息模板**：使用 `{{欄位名稱}}` 代入變數，如：`報到成功！歡迎 {{姓名}}`。
*   **分頁名稱**：若要支援多個分頁，請用「半形逗號」隔開，如：`學員報到, 場次2, 場次3`。
